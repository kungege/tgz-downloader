# tgz-downloader

![Node CI](https://github.com/kungege/tgz-downloader/actions/workflows/test.yml/badge.svg)


## install

```bash
npm install tgz-downloader -g
```


## usage

```bash
tgz-download --lockfilePath=/Path/**/yarn.lock --outputDir=/Path/**/**
```
